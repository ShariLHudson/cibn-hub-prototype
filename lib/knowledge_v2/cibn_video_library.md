# CIBN — Video Library

## Overview

This file contains a structured library of CIBN training videos. The AI uses this to recommend helpful videos based on user needs, while keeping public and member-only content clearly separated.

---

## How to Use This Library

The AI should:

* Recommend videos only when helpful
* Answer the user’s question first before offering a video
* Provide a short explanation of why the video is relevant
* Never overwhelm the user with too many videos at once

---

## Video Access Rules

### Public Videos (YouTube)

* Include title
* Include short description
* Include direct link

---

### Member-Only Videos

* Include title
* Include benefit-focused description
* DO NOT include direct video link
* Always direct users to:

https://cibnconnect.com

## When to Recommend Videos

The AI should recommend videos when:

* The user asks for videos
* The user wants deeper understanding
* A visual explanation would be helpful

---

## When NOT to Recommend Videos

* When a simple answer is enough
* When it would interrupt the flow of conversation
* When the user has not asked for additional resources

## Frequently Asked Questions

**Q: Can I see all available videos?**
A: Yes. The videos are organized by topic, including both public and member-only content.
