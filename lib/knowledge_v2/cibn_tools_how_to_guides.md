# CIBN — Tools How-To Guides

## Overview

This file provides step-by-step instructions for using the CIBN LinkedIn Extension and related tools. The AI uses this file when users ask “how do I do this?” and need clear, actionable steps.

---

## Core Principle

The tools support networking — they do not replace it.

Always remember:

* Relationships come first
* Trust comes before referrals
* Human follow-through determines results

---

# Getting Started

## Extension Setup (High-Level)

1. Install the extension (optimized for Google Chrome)
2. Open the extension from your browser toolbar
3. Link the extension to your registered account
4. Confirm features are active

---

# How To: Set Up a Scoring Matrix

## What It Does

Assigns a score to LinkedIn profiles based on your goals and search criteria.

---

## Steps

1. Open the main dashboard
2. Click the “two heads” icon
3. Set three categories:

   * Geographic positive
   * Geographic negative
   * Keywords / qualifiers

---

## Best Practices

* Use keywords that appear on LinkedIn profiles
* Avoid invisible intent (e.g., “ready to buy”)
* Keep keywords simple and clear

---

## Strategy Options

### Direct Client Targeting

* Targets your ideal clients
* Good for immediate opportunities

---

### Referral Partner Targeting (Recommended)

* Targets people connected to your ideal clients
* Creates more consistent long-term opportunities

---

# How To: Use Match Button

## What It Does

Finds high-likelihood people to meet based on your criteria.

---

## Steps

1. Open Match Button
2. Review suggested profiles
3. Click a profile to evaluate
4. Identify mutual connection
5. Decide if you want to meet
6. Request introduction

---

## Important Rule

If trust is low:

* Reconnect first
* Build relationship
* Then request introductions

---

# How To: Request an Introduction

## Steps

1. Click “Introduction” on profile
2. Enter mutual connection name
3. Select the person
4. Review AI-generated message
5. Personalize slightly
6. Send request

---

## Follow-Up Process

1. Message your connection directly
2. Mention the request
3. Ask if they’re comfortable introducing
4. Explain the purpose

---

# How To: Give a Golden Recommendation

## What It Is

A strong trust signal showing confidence in someone.

---

## Steps

1. Open trusted connection’s profile
2. Use Golden Recommendation tool
3. Write recommendation
4. Send message to notify them

---

## Best Practice

* Only give to people you trust
* Follow up after giving

---
