CIBN System Overview
Purpose of CIBN:
CIBN Connect is a structured networking system designed to help professionals build relationships that produce consistent referrals.
The system focuses on:
• collaboration
• trust-based introductions
• structured follow-up
• measurable networking results
________________________________________
Core Networking Activities
Members focus on:
• attending networking meetings
• conducting one-on-one meetings
• building collaborating partners
• making introductions
• tracking referrals
________________________________________
Collaborating Partners
Members aim to build relationships with 3–5 collaborating partners who:
• serve the same audience
• offer complementary services
• regularly exchange referrals
________________________________________
Networking Tools
The CIBN ecosystem uses several tools.
LinkedIn
Primary networking environment.
Connection Extension
Helps identify aligned connections and request introductions.
Meetn
Hosts networking meetings and online events.
Nolodex
Tracks referrals and referral revenue.
CRM Integration
Allows networking activity to connect with sales pipelines.
